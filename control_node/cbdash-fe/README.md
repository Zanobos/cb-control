# Polar-dashboard

## Prerequisites

[Node.js](https://nodejs.org).

## Installation

```bash
npm install
```

## Development

```bash
npm run serve
```

## Deploy

```bash
npm run build
```

## Customization

The primary colors of the application can be configured through sass variables in `src/assets/sass/colors.scss` .

The sidebar logo is loaded from `src/assets/img/logo.png` .
